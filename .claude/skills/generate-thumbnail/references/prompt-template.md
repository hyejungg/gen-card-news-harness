# 썸네일 헤드라인 생성 프롬프트 구조

아래 구조를 참조하여 카피를 생성한다. 이 파일의 내용을 그대로 사용하는 것이 아니라, 구조와 규칙을 이해하고 입력값에 맞게 카피를 만든다.

---

## 프롬프트에 포함할 요소

1. **서비스 컨텍스트**: service_name, service_desc, brand_label
2. **타겟 정보**: 페르소나, 욕구, 인지단계
3. **5개 앵글 설명**:
   - 공감: 현재 상황/감정에 공감하는 톤
   - 공포: 위기감을 자극하는 톤
   - 이익: 구체적 혜택/숫자를 제시하는 톤
   - 편의: 쉽고 간편함을 강조하는 톤
   - 사회증거: 다른 사람도 하고 있다/전문가 추천 톤

## 규칙

- 각 앵글별 헤드라인 정확히 3줄
- 각 줄 한국어 7~12자 (공백 포함)
- 칩1 = 페르소나 키워드, 칩2 = 콘텐츠 키워드 (# 포함)
- 말투: ~요체 또는 ~다체 중 하나 (혼용 금지)
- 이모지 사용 금지

## 응답 JSON 구조

```json
{
  "angles": [
    {
      "angle": "공감",
      "headline": ["첫째줄", "둘째줄", "셋째줄"],
      "chip1": "#페르소나키워드",
      "chip2": "#콘텐츠키워드"
    }
  ]
}
```
